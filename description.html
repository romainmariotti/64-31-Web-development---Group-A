Project Description

By Romain Mariotti the 17/10/2024 at 09:00

Our game is based on Space Invaders. The player will be playing as a jet plane, dodging through meteors and shooting them, earning points by doing so. The goal of the game is to score as many points as possible in order to reach the top of the leaderboard. The more meteors the player shoots, the more points he earns.

The player has 3 lives. Every time he gets hit by a meteor, the player loses 1 life. When his life reaches 0, the game ends. Every time the player shoots a meteor, he earns 100 points.

The source code of the application is available on the github of the project .
